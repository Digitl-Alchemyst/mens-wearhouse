<h3><ul><b>How to use</b></ul></h3>

## Getting Started

## **Dependencies**

    Next.js 14.0.4
    Tailwind CSS 3.3.2
    Typescript 5.1.6
    React Skeleton Loading 3.1.1

### Install Dependencies

    npm i

### Start Dev Enviroment

    npm run dev

The app will start on [http://localhost:3000](http://localhost:3000) in your browser. 

### Deploy

    vercel
    vercel --prod
